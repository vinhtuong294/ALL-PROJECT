# 🖼 1. DESIGN SPECS (Yêu cầu Thiết kế UI - DNGO Shipper App)

## A. Design System (Tuỳ chỉnh chung)
- **Phông chữ (Font Family):** Roboto (Dùng xuyên suốt từ Headline, Display đến Body/Label)
- **Màu UI Hệ thống (Material 3 - Nút bấm, Thanh điều hướng, Form):**
  - **Light Mode:** Indigo (`#6366F1`) 
  - **Dark Mode:** Indigo Nhạt (`#818CF8`)
- **Màu thương hiệu / Điểm nhấn (DNGO Specific):**
  - **Primary Green (Xanh lá chủ đạo):** `#2F8000`
  - **Light/Card Background (Nền xanh nhạt):** `#D7FFBD`
