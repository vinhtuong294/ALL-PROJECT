# Site Vision
A robust but beautifully simple Seller Dashboard for DNGo, a fresh grocery market app. The dashboard needs to give sellers quick insights into their revenue, pending orders, inventory status, and a toggle to quickly open/close their shop.

## Sitemap
- [ ] seller_home (Dashboard Overview)
- [ ] seller_orders (Order Management)
- [ ] seller_products (Inventory Management)

## Roadmap
1. Build the Seller Home Dashboard.
2. Build the Order Management page.
3. Build the Inventory Management page.

## Creative Freedom
- Think about what a farmer or grocer would need to see quickly without getting bogged down by complicated charts.
- Emphasize clear statuses (New Order, Processing, Shipped).
